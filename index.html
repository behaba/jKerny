jKerny kerning get easy

Powered by the power of Lettering.js and inspiration drawn on Kerningjs.com, jKerny gives you the ability to use brand new pseudo-selectors (well ok... they aren't W3C valid!)

Download jKerny

Installation ?

Just include the latest jquery library and the jKerny plugin and you are right!

Let's take some examples

:first-word & :word:first-child

You want to style the first word and apply a hover effect, no problem! 
p:word:first-child { color:red; }
or 
p:first-word { color:red; } p:first-word:hover { color:yellow; }
produce

Johnny take a walk with your sister the moon

:last-word & :word:last-child

Ok now the last word, no problem! 
p:word:last-child { font-weight:bold; background-color:red; color:#fff; }
or 
p:last-word { font-weight:bold; background-color:red; color:#fff; }
produce

Johnny take a walk with your sister the moon

Guess what, it works also on letter!

:first-letter & :letter:first-child

OK :first-letter is already a W3C specification but what about a cross-browser solution ? 
p:word:first-letter { font-size:2em; font-weight:bold; position:relative; top:0.2em; }
or 
p:first-letter { 	font-size:2em; font-weight:bold; position:relative; top:0.2em; }
produce

Johnny take a walk with your sister the moon

:last-letter & :letter:last-child

You want to target the last letter and there's reason to stop you! 
p:word:last-letter { .example4 p:last-letter { background:lime; color:#fff; border-radius:5px; }
or 
p:last-letter { .example4 p:last-letter { background:lime; color:#fff; border-radius:5px; }
produce

Johnny take a walk with your sister the moon

And what about Kerning ?

-jkerny-letter-spacing

If you want to adjust the space between letter, jKerny let you use a simple property "-jkerny-letter-spacing", let's see! 
p { font-size: 1.5em; -jkerny-letter-spacing: 2px -2px -1px -1px 2px; }
produce

Ah Men!

-jkerny-word-spacing

Hey, you can also adjust word-spacing with the "-jkerny-word-spacing" property! 
p { font-size: 1.5em; -jkerny-word-spacing: -2px -2px -2px -2px; }
produce

The word is mine!

What else?

There is no limit with pseudo-selector, you can combining :word or :letter with all the other existing pseudo-selectors.
Need some ideas ? 
h3:word:first-of-type
strong:letter:nth-child(2n)
p:first-child:word:last-of-type(2n)
